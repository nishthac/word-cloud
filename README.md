- inflect

##### Dev Tool : 
-   Jupyter Notebook

#### Running Locally
- Download Install Jupyter 
- Load the ```.ipynb``` file
- Set correct path for text books and mask images
- Uncomment ```#nltk.download('stopwords')``` and ```#nltk.download('wordnet')``` to install these packages if not already present
